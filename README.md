- `README.md`: Ovaj fajl.

## Projekti

U `src` direktorijumu se nalaze sledeći projekti:

- `Prakticne-1`
- `Prakticne-2`

## Dokumentacija

Sva prateća dokumentacija i uputstva za vežbe se nalaze u `documentation` direktorijumu u PDF formatu.

## Zahtevi

Za otvaranje i rad sa projektima, neophodno je imati instaliran [Siemens TIA Portal](https://support.industry.siemens.com/cs/document/109784440/simatic-step-7-and-wincc-v17-trial-download?dti=0&lc=en-WW).

# Practical Exercises for Siemens S7-1200 PLC

This repository contains a set of practical exercises for programming the Siemens S7-1200 PLC, created in TIA Portal.

## Repository Structure

The project is organized as follows:

- `src/`: Contains all the TIA Portal projects for each exercise.
- `documentation/`: Contains PDF documents with instructions for the exercises.
- `README.md`: This file.

## Projects

The `src` directory contains the following projects:

- `Prakticne-1`
- `Prakticne-2`
- `Prakticne-3_brojac`
- `Prakticne-3_sabirac`
- `Prakticne-4`
- `Prakticne-5`
- `Prakticne-6`
- `Prakticne-setup(Template_projekta)`

## Documentation

All accompanying documentation and exercise instructions can be found in the `documentation` directory in PDF format.

## Requirements

To open and work with the projects, it is necessary to have [Siemens TIA Portal](https://support.industry.siemens.com/cs/document/109784440/simatic-step-7-and-wincc-v17-trial-download?dti=0&lc=en-WW) installed.
